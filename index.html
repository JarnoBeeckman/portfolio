MasterMind

Eerste jaar informatica (begin 2021): Mijn allereerste project als developer (op eigen initiatief).

Dit oud gezelschapsspel werd thuis teruggevonden in een kast. Sindsdien speel ik het graag af en toe. Het is een spel voor 2 spelers, hoewel er maar 1 speler echt speelt. De andere maakt de combinatie die de hoofdspeler moet raden en geeft feedback door de gokken te vergelijken met de oplossing. Als je niet de hoofdspeler bent, zal dit spel dus vrij snel voor verveling zorgen. Toen realiseerde ik mij dat de andere speler perfect kon vervangen worden door een computer.

Op dat moment was ik voor web development nog maar recent geintroduceerd aan JavaScript en had ik nog niet zo'n uitgebreide kennis van HTML en CSS. Bij software development begonnen we net aan JavaFX aangezien we al sinds het eerste semester met Java werkten. Dat is dan ook waarom de GUI zo simplistisch is.
Zo werd het MasterMind project dus gecreëerd met JavaFX.

Training manager

Tweede jaar Development (december 2021): Mijn eerste web dev project (schoolproject).

In het tweede semester van het eerste jaar was ik voor het eerst geintroduceerd aan JavaScript. In het tweede jaar hebben we dit voortgezet door met NodeJS te beginnen. Voor de front-end moesten we React gebruiken en voor de back-end een REST API in NodeJS met Koa. Als database heb ik MySQL gebruikt aangezien we deze aangeleerd zijn in het eerste jaar.

Voor dit project mochten we zelf kiezen wat we maakten. Ik heb me gebaseerd op wat mijn sportclub zou kunnen gebruiken. Een systeem waarbij trainers hun trainingen kunnen bijhouden en kijken naar andere trainers hun opstellingen. Eens dit project af was, vond ik het niet kwalitatief genoeg om aan de club te geven. Dit project heeft dan ook nooit het daglicht gezien. Het heeft wel gediend als een soort van hulpmiddel om mij op te baseren voor het websysteem dat ik later voor de club heb gemaakt. (Zie Dendermondse BC project)

In dit project was authenticatie, validatie in beide front-en back-end en wachtwoord hashing gebruikt. Zo hadden alleen de trainers toegang tot het bewerken en aanmaken van trainingen. Gewone leden konden alleen het overzicht zien.

Duurzaamheid KMO's

Tweede jaar AI & Data Engineering (juni 2022): Het eerste (school)project als Data Engineer.

Voor het vak "Data Engineering Project I" kregen we de opdracht om alle relevante informatie omtrend duurzaamheid binnen vlaamse kmo's te verzamelen. Aan de hand van de verzamelde data werd dan ook een duurzaamheidsscore toegekend aan elke kmo. Hierbij moesten we gebruik maken van "Web Scraping" voor een lijst van ongeveer 80.000 kmo's. Bij het scrapen van de websites en jaarverslagen voor elke kmo, werd al deze informatie automatisch geanalyseerd en gestructureerd opgeslagen in een SQL database. De lijst van alle kmo's en hun informatie hadden we gekregen bij de opdracht van dit project in CSV formaat en het verzamelen van data en opvullen van de database werd met Python gedaan.

Met alleen een database is de klant natuurlijk niet veel. Zo hebben we ook een beveiligde API server gemaakt in NodeJS en een beveiligde website in React om de informatie mooi te weergeven. Ook werd ons project uitverkozen tot beste resultaat door de klant.

Dit project was een groepswerk. Mijn groepsleden waren Siebe Van Campenhout, Benjamin Daems, Guylian Weverbergh en Tanguy Montaine.
